### HOW TO PLAY?
Run ```Zombioo.exe``` or simply open ```MAINGAME.py```

To create .exe file type in terminal:     
```pyinstaller.exe --onefile .\MAINGAME.py```


or use auto py to exe
[PuPI](https://pypi.org/project/auto-py-to-exe/)
